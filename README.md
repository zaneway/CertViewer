# HeTu(河图)
* 河图与洛书是中国古代传说中上天授予的祥瑞之兆.
* 本工程借"河图"之名,用于解析各种密码学相关结构.


# 简单介绍
    本工程使用语言开发, 基于使用 fyne 构建的一个可视化工具.
        当前提供两种模式:
             1. 解析X509证书
             2. 解析Asn1编码(正在完善)
             3. ...(构思中)


# 快速开始

```shell
# 提供了Windows、MacOS的打包脚本,直接执行即可.
sh build-macos-app.sh

sh build-windows-app.sh

```
